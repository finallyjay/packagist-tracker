# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

## Our pledge

We pledge to make participation in our project a welcoming experience for
everyone, and to act in ways that contribute to an open and healthy community.

## Scope

This Code of Conduct applies within all project spaces, and also applies when
an individual is representing the project in public spaces.

## Enforcement

Concerns about conduct in this project can be reported by opening a
confidential issue or contacting the maintainer through GitHub.

For the full text, including examples and enforcement guidelines, see the
[Contributor Covenant website](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
